# crowdsource-smart-home-data

Repository containing the main components of the project, such as the APIs and data lake configuration.

### Run frontend

```bash
docker-compose up --build
```

### Ingest API ui
```
http://localhost:8080/api/ingest/ui/
```

### Grafana ui
```
http://localhost:3000/
```
